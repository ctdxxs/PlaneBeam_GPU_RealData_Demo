minifus Control and Analysis Software Platform

Powerful MATLAB-based Interactive Interface

We provide a specialized control and analysis software for functional ultrasound imaging, developed based on the MATLAB environment, offering users an intuitive graphical user interface (GUI) and real-time processing capabilities. The platform focuses on real-time functional imaging, does not support raw data saving, but provides efficient functional imaging analysis.

Main Features:

  • Real-time functional imaging display and parameter configuration
  • Real-time data processing and analysis
  • Built-in functional ultrasound imaging algorithms (blood flow detection, functional imaging, etc.)
  • Real-time image reconstruction and display
  • Real-time export of functional imaging results
  • Open source GPU real-time library calls, supporting CUDA/OpenCL parallel computing

PlaneBeam_GPU_RealData_MEX Open Source MEX Program

High-Performance GPU Parallel Computing Solution

PlaneBeam_GPU_RealData_MEX is a MEX program we developed for calling real-time imaging GPU dynamic libraries, specifically providing high-performance parallel computing support for minifus platform's functional ultrasound imaging. The program is based on CUDA/OpenCL technology, fully utilizing modern GPU parallel computing capabilities for real-time data processing and image reconstruction.

Open Source Features:

  • Open Source Call Interface: Provides complete MEX call interface source code with custom configuration support
  • Flexible Configuration System: Open source configuration parameters and calling methods for easy user customization
  • MATLAB Integration: Provides MATLAB MEX interface for seamless integration into MATLAB environment
  • Real-time Plane Wave Imaging: Supports real-time image reconstruction in plane wave transmission mode
  • High-performance Optimization: Deep optimization for real-time processing
